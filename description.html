Schelling model

Interact with the widgets to change the parameter values of the model. Clicking the reset button will update the changed values. The simulation will run first until convergence or until the maximum amount of steps is reached. An agent is satisfied (square if satisfied, circle if not) if the fraction of similar neighbours is larger than its homophily value (can be seen by hovering over the agent). For more information please visit the COMPASS website or contact e.p.n.dignum@uva.nl.
